# 脚本工具

## clear-cache.bat

**用途**: 清理开发环境缓存，解决图标/资源不更新问题

**使用方法**:
1. 双击运行 `scripts/clear-cache.bat`
2. 按提示关闭开发服务器
3. 重新运行 `npm run dev`

**清理内容**:
- Vite 开发缓存 (`.vite`)
- 构建缓存 (`.cache`)
- Tauri 应用数据

## 何时使用

- 更新了图标但显示旧图标
- 静态资源（图片、CSS）修改后不生效
- 遇到奇怪的缓存问题

## 注意事项

- 清理后首次启动可能稍慢（需要重新构建缓存）
- 不会影响源代码，只删除临时缓存文件
